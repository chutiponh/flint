  <action>
Create `UI/AllToolsGridView.swift` as a stateless component per D-01. Use `@Environment(ToolRegistry.self) private var toolRegistry` and a prop `let onSelect: (String) -> Void`. Body is a `LazyVGrid(columns: [GridItem(.adaptive(minimum: 100, maximum: 140), spacing: 8)], spacing: 8)` iterating `ForEach(toolRegistry.tools)` (all 12, no filter, registry order — do not reorder). Each tile is a `.plain` `Button` whose action calls `onSelect(tool.id)`, with a `VStack(spacing: 8)` containing `Image(systemName: tool.sfSymbol)` at `.font(.system(size: 22))` in `.accentColor` over `Text(tool.name)` at `.font(.system(size: 13))` regular, `.lineLimit(2)`, `.multilineTextAlignment(.center)`. Tile `.frame(maxWidth: .infinity).padding(.vertical, 8)`, background `.quaternary.opacity(0.5)` with `.cornerRadius(8)`; on hover (via `.onHover` + a per-tile `@State` flag inside a private tile subview) the background tints to `.quaternary.opacity(0.85)`. Outer `.padding(.horizontal, 12).padding(.vertical, 8)`. Each tile: `.accessibilityLabel(tool.name)` and `.accessibilityHint("Open \(tool.name)")`. No `@State` at the grid-view level — navigation state stays in MenuBarPopoverView.

Create `UI/ToolHeaderView.swift` as a stateless component per D-02. Props: `let toolName: String` and `let onBack: () -> Void`. Body: `VStack(spacing: 0)` wrapping an `HStack` then `Divider()`. The HStack contains a leading `.plain` `Button` (action `onBack`) labeled with `Image(systemName: "chevron.left")` + `Text("All Tools")` both in `.accentColor` at `.font(.system(size: 13))`, with `.accessibilityLabel("Back to tool picker")`; a centered `Text(toolName)` at `.font(.system(size: 15, weight: .semibold))` `.foregroundColor(.primary)` with `.accessibilityAddTraits(.isHeader)`; and a trailing invisible spacer matching the back button width to keep the title visually centered. HStack `.frame(minHeight: 44).padding(.horizontal, 12)`. `Divider()` below uses the default separator color. The `Divider()` uses `Color(NSColor.separatorColor)` implicitly via the system Divider — do not apply a custom color.

Both files use only SwiftUI semantic colors (INFRA-14) and supply VoiceOver labels (INFRA-15). No `import AppKit` needed beyond what SwiftUI provides for these display-only views.
