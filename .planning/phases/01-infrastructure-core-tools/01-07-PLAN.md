| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-07-EP | Elevation of Privilege | Release entitlements regression | mitigate | Re-confirm `Lathe-release.entitlements` still has NO `get-task-allow` and Hardened Runtime after Release-config work (T-01-EP carried forward). [BLOCKING] grep check. |
| T-07-DOS | Denial of Service | Cold-start / RAM budget | mitigate | Lazy ViewModel init + off-main GRDB open keep startup under budget (pitfall #6); verified in Task 4 (INFRA-18). |
| T-07-SC | Tampering | SMAppService login-item registration | accept | Apple-sanctioned `SMAppService.mainApp` API; user-initiated; macOS shows a system notification on change. |
</threat_model>
